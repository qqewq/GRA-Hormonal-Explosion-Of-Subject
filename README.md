## What is this?

This repository implements an **aesthetic engine** that takes any large structured corpus — films, scientific theories, cultural archives — and applies GRA (Gödel–Reductio ad Absurdum) nullification to produce a **combined masterpiece** that triggers a "hormonal explosion": strong emotional and aesthetic resonance, while remaining hierarchically stable.

The style is inspired by *What Ever Happened to Baby Jane?* — we preserve local "madness" (subjective foam) but nullify global chaos, resulting in a structured, yet overwhelming, work of art.

## Core idea

1. **Semantic Foam** – any domain contains contradictions, noise, and narrative dissonance.
2. **GRA nullification** – measure global foam $\Phi$, iteratively reduce it while keeping local foam alive.
3. **Hormonal index $H$** – maximize affective intensity (fear, joy, catharsis, shock).
4. **Orchestra of critics** – role‑based agents (Critic, Director, Composer) collaborate to sculpt the output.

## Architecture

- `semantic_pipeline/` – ingest data, build embeddings, compute foam metrics.
- `gra_integration/` – adapter to GRA‑Core, Rank‑N stability hooks.
- `orchestra/` – multi‑agent swarm that nullifies global foam while preserving local madness.
- `output/` – generated playlists, scenarios, meta‑maps of the explosion.
